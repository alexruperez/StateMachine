StateMachine

StateMachine is a Swift library to State machine Swift framework inspired by GKStateMachine from Apple GameplayKit framework manage the task execution in different threads. Through the definition a simple but powerful concept, StateMachine.

üì≤ Installation

StateMachine is available through CocoaPods. To install it, simply add the following line to your Podfile:

pod 'ArchitStateMachine'

Or you can install it with Carthage:

github "alexruperez/StateMachine"

Or install it with Swift Package Manager:

dependencies: [
    .package(url: "https://github.com/alexruperez/StateMachine.git")
]
